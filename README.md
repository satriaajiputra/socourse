# socourse
A simple custom UI - Java GUI with Swing Framework

## Quick Installation
1. Clone the repository
```bash
git clone https://github.com/satriaajiputra/socourse.git
```
2. Open project using ``netbeans 8.2``
3. Run project by pressing ``F6``
4. Happy coding
